# Java Notes App (File I/O)

## 📌 Objective

A simple text-based Notes Manager using Java File I/O.

## 🛠 Tools Used

* Java
* VS Code
* Terminal

## 🚀 Features

* Add notes
* View saved notes
* Persistent storage using file handling

## 📂 File Handling

* Uses FileWriter to store notes
* Uses BufferedReader to read notes

## ▶️ How to Run

```bash
javac NotesApp.java
java NotesApp
```

## 📁 Output File

* Notes are stored in `notes.txt`

## 💡 Concepts Used

* FileWriter
* FileReader
* BufferedReader
* Exception Handling
* Loops & Switch Case
